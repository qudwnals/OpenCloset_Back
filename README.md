OpenCloset 어플리케이션은 P2P 사용자 주소 기반 서비스를 통해,이웃 간 의류 대여를 가능하게 합니다.
사용자는 직접 입어보지 못했던 옷이나 입어보고 싶은 옷을이웃에게 빌려 입어보는 경험을 통해 새로운 스타일을 시도하고
자신의 욕구를 충족할 수 있습니다.


## 제작 기간 
프로젝트 기간:2025.03.20 - 2025.06.24
